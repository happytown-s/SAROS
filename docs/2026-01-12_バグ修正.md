# 2026-01-12 バグ修正録

## オーディオ入力のルーティング修正

*   **症状**: 入力音声が検知されず、トリガーやモニタリングが機能しない。
*   **原因**: `MainComponent` クラスにて `juce::AudioDeviceManager deviceManager;` が再定義（シャドウイング）されていた。
*   **解決**: 重複定義を削除し、親クラス `AudioAppComponent` のインスタンスを統一して使用。

## FX ON/OFF トグルの修正

*   **症状**: 一部のエフェクト（Flanger, Chorus等）がUIから切り替わらない。また Granular Cloud がMIDIから反応しない。
*   **原因**: `FXPanel.cpp` 内のトグル処理に、特定エフェクトタイプの `switch` ケースが欠落していた。
*   **解決**: すべてのエフェクトタイプ（Flanger, Chorus, Tremolo, Slicer, Bitcrusher, GranularCloud）を処理ルートに追加。

## VS Code デバッグ環境の整備

*   **内容**: macOS環境で `F5` による C++ デバッグ（CodeLLDB）を有効化し、ビルドと実行を自動化。
