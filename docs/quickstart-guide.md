## 1. 設定画面を開く

メイン画面下部の **⚙ SETUP** ボタンをクリックして設定ウィンドウを開きます。

設定ウィンドウには2つのタブがあります：

- **Device** - オーディオインターフェースの選択
- **Trigger** - 自動録音トリガーの設定

---

## 2. オーディオデバイスの設定

### Deviceタブで設定

1. **Output** セクションでスピーカー/ヘッドフォン出力を選択
2. **Input** セクションでマイク/楽器入力を選択
3. **Sample Rate** を選択（48000 Hz 推奨）
4. **Buffer Size** を調整（256〜512 推奨）

> 💡 **ヒント**: バッファサイズが小さいほどレイテンシーが低くなりますが、CPU負荷が増えます。

---

## 3. 入力レベルの確認

### Triggerタブで確認

1. **Trigger** タブをクリック
2. 画面下部の **Input Channel Details** セクションでチャンネルごとのレベルメーターを確認
3. 入力音がメーターに反映されていることを確認

| メーター色 | 状態 |
| ---------- | ------ |
| シアン（水色） | 通常入力 |
| 赤 | 閾値超過（トリガー発動中） |

---

## 4. ステレオリンクの設定

ステレオペア（1-2、3-4など）をリンクして、どちらかのチャンネルが閾値を超えたら両方でトリガーを発動させます。

### 設定方法

1. **Trigger** タブを開く
2. 各チャンネルペアの **LINK** ボタンをクリック
3. ボタンがマゼンタ（紫）に点灯 = リンク有効

### チャンネル個別のオン/オフ

- 各チャンネルの **ON** ボタンでトリガー検知の有効/無効を切り替え
- 使用しないチャンネルはオフにすることでノイズによる誤検知を防止

---

## 5. 自動検知閾値の設定

音量が閾値を超えると自動的に録音が開始されます。

### 方法1: 手動設定

1. **Trigger** タブの **Sensitivity** スライダーを調整
2. マスターメーター（スライダー下の横バー）で白い閾値ラインを確認
3. 演奏時にメーターが閾値ラインを超えることを確認

### 方法2: 自動キャリブレーション

環境ノイズを自動測定して最適な閾値を設定します。

1. **Use Auto Calibration** ボタンをオン（緑色に点灯）
2. **Run Calibration (2s)** ボタンをクリック
3. 2秒間、演奏せずに待機（環境ノイズを測定）
4. 自動的にノイズフロアに基づいた閾値が設定される

> ⚠️ **注意**: キャリブレーション中は演奏しないでください。

---

## 6. 自動録音の開始

設定が完了したら、SAROSは自動的に音声を検知して録音を開始します。

### 録音フロー

1. 設定画面を閉じてメイン画面に戻る
2. 録音したいトラック（1〜8）をクリックして選択
3. 楽器を演奏開始
4. 閾値を超えた瞬間に自動的に録音開始
5. 録音を停止するには **⏺ REC** ボタンをクリック

### ルックバック機能

SAROSは常に入力を監視しており、トリガー発動前の音声も自動的にキャプチャします。これにより、最初の音の頭が切れることを防ぎます。

---

## 7. 手動録音の操作

自動録音を使わずに手動で録音することも可能です。

### 手動録音の手順

1. トラッククリックで選択
2. **⏺ REC** ボタンをクリックして録音開始
3. 演奏する
4. 再度 **⏺ REC** ボタンをクリックして録音停止

### その他の操作

| ボタン | 機能 |
| ------- | ------ |
| **▶ PLAY** | 全トラック再生/停止 |
| **↶ UNDO** | 直前の録音を取り消し |
| **⌫ CLEAR** | 全トラッククリア |

---

## 💡 トラブルシューティング

### 音が入力されない

- Deviceタブで正しい入力デバイスが選択されているか確認
- オーディオインターフェースのゲインを上げる

### 誤検知で録音が始まってしまう

- Sensitivity スライダーを右に動かして閾値を上げる
- 使わないチャンネルのONボタンをオフにする
- Auto Calibration を実行してノイズフロアを測定

### 音の頭が切れる

- ルックバック機能により自動的に対処されますが、極端に短いアタックの場合は手動録音を使用してください
